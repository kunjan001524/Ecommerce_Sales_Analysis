# Ecommerce_Sales_Analysis
Brazilian E-commerce Sales Analysis using Python, Google Colab, and Power BI. This project includes data cleaning, exploratory data analysis (EDA), 6+ visualizations, business insights, and an interactive dashboard.

## Project Overview
This project analyzes the Brazilian E-commerce dataset using Python, Google Colab, and Power BI. The objective is to clean the data, perform exploratory data analysis (EDA), identify business trends, and create an interactive dashboard for better decision-making.

## Objectives
- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Create 5+ visualizations
- Generate business insights
- Build an interactive Power BI dashboard

## Business Insights
1. Most orders were placed from São Paulo.
2. Credit Card was the most preferred payment method.
3. Customer ratings were mostly positive (4–5 stars).
4. Some product categories generated significantly higher sales than others.
5. Sales showed seasonal variations across different months.

## Most Surprising Finding
Although many customers gave excellent ratings, some high-rated product categories did not generate the highest sales. This indicates that customer satisfaction alone does not always lead to higher revenue, and factors such as pricing, marketing, and product visibility also play an important role.
